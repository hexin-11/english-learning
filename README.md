# English Learning

可点击朗读的英语学习网页，包含单词、音标、中文意思和文章朗读。

## 使用

直接打开 `index.html`，或通过 GitHub Pages 发布。

## GitHub Pages

1. 在 GitHub 创建一个名为 `english-learning` 的公开仓库。
2. 上传本项目全部文件。
3. 打开仓库 `Settings → Pages`。
4. 在 `Build and deployment` 中选择 `Deploy from a branch`。
5. 选择 `main` 分支和 `/ (root)`，点击保存。

发布后网址通常为：

`https://hexin-11.github.io/english-learning/`
